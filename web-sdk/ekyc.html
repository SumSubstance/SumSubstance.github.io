      # e-KYC

      WebSDK allows you to create an e-KYC verification step in which an applicant can pass a Non-doc Verification without uploading documents. The system authomatically retrieves all necessary applicant data from the official sources, making the process more simple and secure.

      # Configuration

      Please contact support to activate e-KYC for you
