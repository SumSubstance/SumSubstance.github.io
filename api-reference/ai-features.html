# Liveness and Age Estimation

This method takes a single image as an input and returns liveness and age estimation check results.

POST /resources/imageCapturing/liveness/test

# REQUEST HEADERS
Name Type Value
Content-Type String multipart/form-data
# REQUEST ARGUMENTS
Name Type Required Description
levelName String Yes Name of new user verification steps configuration from the dashboard.
# FORM DATA
Name Type Required Description
content Binary Yes A selfie of a person
metadata Object No Request metadata
# REQUEST metadata BODY PART FIELDS
Name Type Required Description
externalUserId String No An applicant ID on the client side. Must be unique
# RESPONSE

JSON representing check results.

Name Type Description
createdApplicantId String ID of the newly created applicant.
estimatedAgeResult Object Age estimation result.
livenessResult Object Face liveness check result.
# estimatedAgeResult ATTRIBUTES
Name Type Description
age Float Estimated age in years.
# livenessResult ATTRIBUTES
Name Type Description
answer String Liveness check answer (GREEN/RED/ERROR).
errorCode String Reason for ERROR answer.
# Available values of answer
Name Type Description
GREEN String Image passed liveness check. The photo is genuine.
RED String Liveness is suspected to be fraudulent.
ERROR String Image is not suitable for liveness check.
# Available values of errorCode
Name Description
lowQuality Low quality image was provided.
noFace Image does not contain a face.
multipleFaces Image contains several faces.
faceNotAligned Face is cropped, tilted, too large, or too small
# Example request
Copy
curl -X POST \
  'https://api.sumsub.com/resources/imageCapturing/liveness/test' \
  -H 'Content-Type: multipart/form-data' \
  -F 'content=@/SumSub/Example/faceImage.jpg'
  -F 'metadata={"externalUserId": "someUniqueUserId"}'
# Example response
Copy
{
    "createdApplicantId": "64e45a902ca5003993a89561",
    "livenessResult": {
        "answer": "GREEN"
    },
    "estimatedAgeResult": {
        "age": 29.0
    }
}
